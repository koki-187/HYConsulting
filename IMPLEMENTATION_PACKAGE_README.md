## 📋 実装時の重要なポイント

### デザインの統一

- カラーパレット: メインカラー `#0D274D`、サブカラー1 `#008E7D`
- フォントファミリー: 既存のグローバルスタイルと統一
- ボーダーラジアス: 14px（セクション）、8px（フォームフィールド）

### レスポンシブ対応

- デスクトップ: 2列グリッド（1fr 1fr）
- モバイル: 1列グリッド（1fr）
- ブレイクポイント: 768px

### iOS対応

- 入力フォームのフォントサイズは16px以上（自動ズーム防止）

### SEOキーワード

- 「横浜」「湘南」がセレクトボックスのオプションに含まれている

## 🔧 トラブルシューティング

### セレクトボックスが追加できない場合

- STUDIOの「フォーム」機能を使用していることを確認
- フォームコンポーネントが正しく配置されているか確認

### グリッドレイアウトが崩れる場合

- グリッドコンテナの「ディスプレイ」が `Grid` に設定されているか確認
- グリッドテンプレートカラムが正しく設定されているか確認

### モバイル表示で1列にならない場合

- レスポンシブ設定で、モバイルブレイクポイントのグリッドテンプレートカラムを `1fr` に設定
