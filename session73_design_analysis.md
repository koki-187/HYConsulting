# HYConsulting公式HP 背景デザイン分析

## 確認日: 2026-01-09

## 1. Aboutセクション
- **大きな"H"のウォーターマーク**: イラスト背面に巨大な淡いグレー/ブルーの"H"文字
- 位置: イラストの背面中央、左右端からはみ出す
- 色: 非常に薄い淡色（ほぼ白に近いグレー/ブルー）
- 不透明度: 約3-5%程度

## 2. サービスセクション
- **円形のパートナーリング**: 中央にHY Consultingロゴ、周囲に各パートナー（税理士、司法書士、弁護士等）の円形アイコン
- **巨大な"HY CONSULTING"テキスト**: リング背面に全幅を覆う大きなテキスト
  - 色: 非常に薄いブルー/グレー
  - サイズ: 画面幅の180-200%
  - 不透明度: 約3-4%
- **サービスカード背面のパステル図形**: 各カード（01-04）の背面に角丸のパステル図形
  - 形: 角丸四角や流線形
  - 色: #F2F7FD〜#F0F5FA
  - 不透明度: 6-8%

## 3. ヒーローセクション
- 青い球体（バブル）のアニメーション
- 都市イラスト（アイソメトリック）
- 背景は白/薄いグレー

## 4. コンタクトセクション
- 青いグラデーション背景
- 白いテキスト

## 実装指針
1. z-indexは最背面に設定
2. 不透明度は3-5%の範囲
3. カラーは背景色より少し明るいブルー系（#EEF5FC、#F2F7FD）
4. レスポンシブ対応必須
5. テキストやCTAボタンの可読性を優先
