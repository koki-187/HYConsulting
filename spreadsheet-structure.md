# Google Sheets 構造確認

## 無料不動産査定タブ

### 列構造（A〜O列）
| 列 | ヘッダー名 | 備考 |
|---|---|---|
| A | 受付日時 | |
| B | お名前 | |
| C | メールアドレス | |
| D | 電話番号 | 現在は数値として表示（0が消える問題あり） |
| E | 物件種別 | マンション、戸建て、土地、アパート |
| F | 都道府県 | |
| G | 市区町村 | |
| H | 所在地 | |
| I | 床面積（㎡） | |
| J | 築年数（年） | |
| K | 推定価格（万円） | |
| L | 最寄り駅 | |
| M | 駅徒歩（分） | |
| N | 建築構造(アパートのみ) | **新規追加** |
| O | 階建て(アパートのみ） | **新規追加** |

### 確認済みデータ
- 行2〜7: 戸建て、マンション、土地、アパートのデータあり
- 行8〜12: 山田太郎様のアパートデータ（東京都港区）
  - 床面積: 250㎡
  - 築年数: 15年
  - 推定価格: 11,537万円〜15,376万円
  - 最寄り駅: 麻布十番駅
  - 駅徒歩: 3分
  - **建築構造（N列）: 空欄** ← 要修正
  - **階建て（O列）: 空欄** ← 要修正

### 問題点
1. **電話番号（D列）**: 数値として保存されているため、先頭の0が消えている
   - 例: 8012345678 → 正しくは 08012345678
   - 対策: 電話番号を文字列として送信する

2. **アパート専用項目（N・O列）**: 
   - 建築構造と階建てが空欄になっている
   - サーバー側でこれらの項目をGoogle Sheetsに送信する処理を追加する必要がある
