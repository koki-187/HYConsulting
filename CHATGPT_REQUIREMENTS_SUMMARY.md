# HY Consulting LP - ChatGPT共有ドキュメント要件まとめ

## 1. 不動産分析データの構成

### 入力項目（検索フォーム）
- **物件種別**: マンション、戸建て、アパート、土地
- **都道府県**: 19都道府県（五十音順）
- **市区町村**: 1,852市区町村（都道府県と連動）
- **面積**: ㎡
- **築年数**: 年（※土地は不要）

### 出力される3つのレポート
1. **市場分析レポート**
   - 周辺相場
   - 取引件数
   - 平均坪単価
   - 空室率

2. **価値評価レポート**
   - 推定価格
   - 価格レンジ
   - 評価ポイント
   - 投資価値

3. **将来予測レポート**
   - 1年/3年/5年後の予測価格
   - その根拠

### データソース
- 国土交通省の実データ約10万件
- 19都道府県・1,852市区町村の取引データをカバー

## 2. LP全体の設計方針

### ブランド軸
- **誠実 × 専門性**
- タグライン: 「負の不動産を、富と安心の不動産へ。」
- サブコピー: 「横浜・湘南エリアの不動産ドクターが、国交省データ×AI分析で、あなたの不動産の今と未来を可視化します。」

### 事業軸（3つの入口）
1. 相続・実家じまい
2. 空き家・負動産活用
3. 事故物件・訳あり物件

### LP構成セクション
1. **ヒーローセクション（ファーストビュー）**
   - タグライン
   - サブコピー
   - CTAボタン: 「30秒で価値を診断する（無料）」
   - 下部に3つの入口カード

2. **不動産AI分析セクション（メイン）**
   - 説明テキスト + 分析ウィジェット（Embed）
   - 「入力 → 分析 → 3つのレポート」が一目で分かるUI

3. **事例＆お客様の声セクション**
   - Before / Approach / After / Voice の4段構成
   - 最低3件以上

4. **専門家ネットワーク・地域の最強チーム**
   - 顔写真 + 肩書き + 一言コメント
   - 「HYがハブになり、士業・工務店・清掃業者・リノベ会社が動く」図解

5. **無料相談／会員登録CTA**
   - 入力項目は最小限（メール + 名前）

## 3. 技術アーキテクチャ

### 元々の構成
- **フロント**: React（AIReportDemo.jsx）
- **バックエンド**: Node.js + Express（real-estate-data.ts）
- **DB**: JSONファイル（realEstateDataByType.json）

### STUDIO連携時の構成（2層構成）

#### 層1: 分析API & ウィジェット（開発者側で準備）
- real-estate-analysis-exportを使用
- APIエンドポイント:
  - `/api/real-estate-data/prefecture-map`
  - `/api/real-estate-data/calculate-price`
- 本番環境にデプロイ
- ReactのAIReportDemoをビルド
- 公開URL: `https://analysis.hyconsulting.jp/` など

#### 層2: STUDIOサイト
- Embedボックス / iframe で上記ウィジェットを埋め込み
- 必要に応じてSTUDIOのフォーム結果を外部APIへ送る連携

## 4. 運用方法の重要ポイント

### ブランド設定（STUDIO）
- **ベースカラー**: ディープネイビー + 白（信頼×専門性）
- **アクセント**: エメラルドグリーン（再生・湘南）、ウォームグレー（安心）
- **フォント**: 読みやすいゴシック系 + 見出しをやや太めに

### LPページのベースコピー整理
1. **1行キャッチ**
   - 「不動産のここに相談したいいわかりがない」 → 「丸ごと引き受けるワンストップ相談窓口」

2. **サブコピー**
   - 「売る・買うだけでなく、法律・税金・ライフプランまで。全国の専門家ネットワークとともに、あたに最適な選択肢を一緒に考えます。」

### 3つの事業ブロック
1. **不動産購入・売却・活用支援**
   - ベースになる要素（原稿から）
   - 「どこに相談したいいわかりがない」
   - 「日本全国・複数の不動産でも窓口を一本化したい」
   - 「売る、買うだけでなく、法律や税金も含め総合的にアドバイス欲しい」
   - 「丸ごとHY Consultingにお任せください」
   - 「無いいご書情・ライフプランを起点とした包括的で柔軟な選択肢の提案」
   - 「日本全国の税理士、弁護士、司法書士、FPのネットワーク」

2. **後継金・小書・相続の整活支援**
   - （セクション2に続く）
