# 査定信頼度（Confidence Score）- 詳細ガイド

## 📊 信頼度とは

査定信頼度は、HY Consulting の不動産査定システムが提供する査定結果の**信頼性・確実性を示す指標**です。

**0～100% のスケール**で表示され、数値が高いほど査定結果の信頼度が高いことを意味します。

---

## 🔍 信頼度の算出基準

### 1. **比較対象物件数（Comparable Count）**

査定計算に使用した比較対象物件の数が信頼度に大きく影響します。

| 比較対象数 | 信頼度への影響 | 説明 |
|----------|-------------|------|
| 3～5 件 | 基本スコア 60% | 最小限のデータ |
| 6～10 件 | 基本スコア 70% | 標準的なデータ量 |
| 11～20 件 | 基本スコア 80% | 十分なデータ量 |
| 21～50 件 | 基本スコア 85% | 豊富なデータ |
| 50 件以上 | 基本スコア 90% | 非常に豊富なデータ |

**計算式:**
```
基本信頼度 = 60 + (比較対象数 - 3) × 2.5
最大値 = 90%
```

**具体例:**
- 5 件の比較対象 → 60 + (5-3) × 2.5 = 65%
- 10 件の比較対象 → 60 + (10-3) × 2.5 = 77.5% → 78%
- 20 件の比較対象 → 60 + (20-3) × 2.5 = 92.5% → 90%（上限）

### 2. **データの新鮮度（Data Recency）**

比較対象物件の取引日が最近であるほど信頼度が高くなります。

| 取引時期 | 調整 | 説明 |
|--------|------|------|
| 直近 3 ヶ月 | +5% | 最新データ |
| 3～6 ヶ月 | +3% | 比較的新しい |
| 6～12 ヶ月 | +1% | 標準的 |
| 1～2 年 | -2% | やや古い |
| 2 年以上 | -5% | 古いデータ |

**具体例:**
- 基本信頼度 75% + 直近 3 ヶ月のデータ調整 (+5%) = **80%**
- 基本信頼度 75% + 1～2 年前のデータ調整 (-2%) = **73%**

### 3. **データのばらつき（Data Variance）**

比較対象物件の価格がどの程度ばらついているかを示します。

**四分位範囲（IQR: Interquartile Range）を使用:**

```
IQR = Q3（第 3 四分位数） - Q1（第 1 四分位数）

ばらつき指数 = IQR / 中央値

ばらつき指数が小さい = データが集中している = 信頼度が高い
ばらつき指数が大きい = データが分散している = 信頼度が低い
```

| ばらつき指数 | 調整 | 説明 |
|----------|------|------|
| 0～0.1 | +5% | 非常に集中 |
| 0.1～0.2 | +3% | 集中している |
| 0.2～0.3 | 0% | 標準的 |
| 0.3～0.5 | -3% | やや分散 |
| 0.5 以上 | -5% | 大きく分散 |

**具体例:**
```
比較対象物件の価格: ¥80M, ¥85M, ¥90M, ¥95M, ¥100M

Q1 = ¥85M
Q3 = ¥95M
IQR = ¥10M
中央値 = ¥90M

ばらつき指数 = 10M / 90M = 0.111 → +3% 調整
```

### 4. **調整要因の妥当性（Adjustment Validity）**

査定計算に適用される調整要因（築年数、駅距離など）の妥当性が信頼度に影響します。

| 調整要因 | 基準 | 信頼度への影響 |
|--------|------|-------------|
| 築年数調整 | ±10 年以内 | 適切 |
| 築年数調整 | ±10 年以上 | -5% |
| 駅距離調整 | ±10 分以内 | 適切 |
| 駅距離調整 | ±10 分以上 | -3% |
| 面積調整 | ±30% 以内 | 適切 |
| 面積調整 | ±30% 以上 | -5% |

**具体例:**
- 入力物件の築年数: 10 年
- 比較対象の平均築年数: 8 年
- 差分: 2 年（±10 年以内） → 調整は適切 → ペナルティなし

## 📈 信頼度の計算例

### **例 1: 東京都渋谷区の戸建て査定**

**入力データ:**
- 物件種別: 戸建て
- 所在地: 東京都渋谷区
- 築年数: 10 年
- 駅距離: 5 分

**計算プロセス:**

```
ステップ 1: 比較対象物件の検索
- 同一市区町村で同一物件種別の取引: 15 件

ステップ 2: 基本信頼度の計算
基本信頼度 = 60 + (15 - 3) × 2.5 = 60 + 30 = 90%
（ただし上限は 90%）

ステップ 3: データ新鮮度の調整
- 比較対象の取引時期: 直近 2 ヶ月
- 調整: +5%
- 現在の信頼度: 90% + 5% = 95% → 上限 90%

ステップ 4: データばらつきの調整
- IQR = ¥8M、中央値 = ¥80M
- ばらつき指数 = 8M / 80M = 0.1
- 調整: +3%
- 現在の信頼度: 90% + 3% = 93% → 上限 90%

ステップ 5: 調整要因の妥当性確認
- 築年数差: 2 年（適切） → ペナルティなし
- 駅距離差: 3 分（適切） → ペナルティなし

最終信頼度: 90%
```

## 💡 信頼度が低い場合の理由と対策

### **理由 1: 比較対象物件が少ない**

**原因:**
- 地方の物件
- 特殊な物件タイプ
- 市場流動性が低い地域

**対策:**
- より広い地域での検索を実施
- 物件タイプの条件を緩和
- 専門家への相談

### **理由 2: 入力データが比較対象と大きく異なる**

**原因:**
- 築年数が大きく異なる（±10 年以上）
- 駅距離が大きく異なる（±10 分以上）
- 面積が大きく異なる（±30% 以上）

**対策:**
- 入力データの正確性を確認
- より詳細な物件情報を入力
- 専門家による詳細査定を依頼

### **理由 3: データのばらつきが大きい**

**原因:**
- 市場の価格変動が大きい
- 物件の個別性が高い
- 市場が不安定

**対策:**
- 複数の査定を比較
- 市場動向の確認
- 専門家への相談

## 📊 信頼度と査定精度の関係

### **統計的な精度**

| 信頼度 | 推定精度 | 説明 |
|--------|--------|------|
| 90～100% | ±5～10% | 非常に正確 |
| 75～89% | ±10～15% | 正確 |
| 60～74% | ±15～25% | 中程度 |
| 50～59% | ±25～35% | 低い |
| 50% 未満 | ±35% 以上 | 参考値のみ |

**具体例:**
```
推定価格: ¥100,000,000
信頼度: 85%（推定精度 ±10～15%）

実際の価格範囲:
¥85,000,000 ～ ¥115,000,000
```

## 🔧 フロントエンド表示実装

### **React コンポーネント例**

```typescript
interface ConfidenceProps {
  confidence: number;
  showLabel?: boolean;
}

export function ConfidenceMeter({ confidence, showLabel = true }: ConfidenceProps) {
  const getColor = (score: number) => {
    if (score >= 90) return "#4CAF50"; // 緑
    if (score >= 75) return "#2196F3"; // 青
    if (score >= 60) return "#FF9800"; // オレンジ
    if (score >= 50) return "#F44336"; // 赤
    return "#D32F2F"; // 濃赤
  };

  const getMessage = (score: number) => {
    if (score >= 90) return "十分なデータに基づいた査定です。信頼性が高いと言えます。";
    if (score >= 75) return "標準的なデータ量に基づいた査定です。";
    if (score >= 60) return "データが限定的ですが、参考値として有効です。";
    if (score >= 50) return "データが不足しています。詳細査定をお勧めします。";
    return "参考値としてのみご利用ください。専門家への相談を強くお勧めします。";
  };

  return (
    <div className="confidence-meter">
      {showLabel && (
        <div className="confidence-label">
          査定信頼度: {confidence}%
        </div>
      )}
      <div className="confidence-bar">
        <div
          className="confidence-fill"
          style={{
            width: `${confidence}%`,
            backgroundColor: getColor(confidence),
          }}
        />
      </div>
      <p className="confidence-message">{getMessage(confidence)}</p>
    </div>
  );
}
```

## 🔐 信頼度の信頼性

### **信頼度計算の透明性**

HY Consulting の信頼度は以下の要因に基づいて**客観的に**計算されます：

1. **比較対象物件数** - 統計的サンプルサイズ
2. **データの新鮮度** - 市場の現在性
3. **データのばらつき** - 統計的分散
4. **調整要因の妥当性** - 物件特性の適合性

### **信頼度の限界**

信頼度が高い場合でも、以下の要因により実際の査定価格が異なる可能性があります：

- 物件の個別的な状態（リフォーム、損傷など）
- 市場の急激な変動
- 特殊な取引条件
- 相続・空き家・事故物件などの特殊性

## 📞 信頼度に関する Q&A

### **Q: 信頼度 50% の場合、査定は信用できませんか？**

**A:** 信頼度 50% でも参考値として有効ですが、より詳細な査定を推奨します。特に取引予定がある場合は、専門家への相談をお勧めします。

### **Q: 信頼度を上げるにはどうすればよいですか？**

**A:** 以下の方法で信頼度が向上する可能性があります：
- より詳細な物件情報を入力
- 駅距離、築年数などの正確な情報を確認
- 市場データが豊富な地域での査定

### **Q: 信頼度は時間とともに変わりますか？**

**A:** はい。新しい取引データが追加されると、比較対象物件数やデータの新鮮度が変わり、信頼度が変動する可能性があります。
