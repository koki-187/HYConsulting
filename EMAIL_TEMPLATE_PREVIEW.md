## 📧 メール送信の流れ

```
┌─────────────────────────────────────────────────────┐
│  ユーザーが査定フォームを入力                          │
│  ✓ 物件種別：戸建て                                  │

│  ✓ メールアドレス：user@example.com（入力）          │

                        ↓
┌─────────────────────────────────────────────────────┐
│  フォーム送信                                        │
│  「査定を依頼する」ボタンをクリック                  │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│  バックエンド処理                                    │
│  1. 査定計算実行                                     │
│  2. データベース保存                                 │
│  3. メールアドレス確認                               │

                        ↓
┌─────────────────────────────────────────────────────┐
│  メール送信実行                                      │
│  EmailService.sendAssessmentEmail()                 │
│  ↓                                                  │
│  メールテンプレート生成                              │
│  ↓                                                  │
│  Nodemailer で送信                                  │
└─────────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────────┐
│  ユーザーがメール受信                                 │
│  📧 user@example.com に到着                         │
└─────────────────────────────────────────────────────┘
```

## 🔧 メール送信の技術仕様

### **SMTP 設定**

```typescript
// 開発環境例
{
  host: "smtp.gmail.com",
  port: 587,
  secure: false,
  auth: {
    user: "your-email@gmail.com",
    pass: "your-app-password"
  }
}

// 本番環境推奨
{
  host: "smtp.sendgrid.net",
  port: 587,
  secure: false,
  auth: {
    user: "apikey",
    pass: "SG.xxxxxxxxxxxxx"
  }
}
```

### **メールヘッダー**

```
From: HY Consulting <noreply@hy-consulting.jp>
To: user@example.com
Subject: 不動産査定結果 - 東京都渋谷区渋谷1-2-3
Content-Type: text/html; charset=UTF-8
Content-Transfer-Encoding: 7bit
Reply-To: noreply@hy-consulting.jp
```

### **メール本体**

```
Content-Type: multipart/alternative

Part 1: text/plain
- プレーンテキスト版（テキストメールクライアント用）

Part 2: text/html
- HTML 版（HTML 対応メールクライアント用）
```

## 📞 サポート情報

### **メール送信に関する問題**

| 問題 | 確認項目 |
|------|--------|
| メール受信されない | SMTP 設定、ファイアウォール、迷惑メール |
| 文字化け | エンコーディング設定（UTF-8） |
| 送信が遅い | ネットワーク接続、SMTP サーバー負荷 |
| スパム判定 | SPF/DKIM 設定、メール内容 |
