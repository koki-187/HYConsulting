# 公式サイト（HP）とLP デザイン比較分析

## 1. 全体構成の違い

### 公式サイト（https://hyconsulting.jp/）
- **ターゲット**: コーポレートサイト、企業情報中心
- **主要セクション**: ABOUT → SERVICES（4つのサービス） → COLUMN → NEWS → CONTACT
- **ナビゲーション**: 会社情報、お知らせ、お役立ち情報、無料相談する
- **コンテンツ**: 企業紹介、サービス概要、ニュース、コラム

### LP（現在のLP）
- **ターゲット**: リード獲得、不動産査定フォーム中心
- **主要セクション**: Hero → Services（3つの柱） → Assessment Form → Features → Case Studies → Testimonials → FAQ → Contact
- **ナビゲーション**: ホーム、サービス、無料査定、強み、実績、よくある質問、お問い合わせ
- **コンテンツ**: 査定フォーム、詳細なサービス説明、事例紹介、FAQ

### サービス構成の差異

| 項目 | 公式サイト | LP |

| 01 | 不動産購入・売却・活用支援 | 老後資金・介護・相続の終活支援 |
| 02 | 老後資金・介護・相続の終活支援 | 不動産購入・売却・活用支援 |
| 03 | ０円物件・負動産の処分活用支援 | ０円物件・負動産の処分活用支援 |

**指摘**: LPでは「不動産パーソン育成採用支援」が削除されている。これは意図的な変更と思われるが、確認が必要。

### ナビゲーション構成

**公式サイト**:
- 会社情報
- お知らせ
- お役立ち情報
- 無料相談する

**LP**:
- ホーム
- サービス
- 無料査定
- 強み
- 実績

## 3. デザイン要素の比較

### ロゴ
- **公式サイト**: アイコン + テキスト「HY Consulting」（左上、白背景パネル内）
- **LP**: アイコン + テキスト「HY Consulting」（ヘッダー・フッター）
- **状態**: ✅ 統一済み

### カラースキーム
- **公式サイト**: 
  - プライマリ: ブルー系（#005BAC相当）
  - セカンダリ: ライトブルー、グレー
  - アクセント: 黒（ボタン）

- **LP**: 
  - プライマリ: ブルー系（#005BAC）
  - アクセント: シアン（#00A0E9）
  - 背景: ホワイト、ライトグレー
  
- **状態**: ✅ 基本的に統一されている

### フォント
- **公式サイト**: Noto Sans JP（日本語）、英字フォント
- **LP**: Noto Sans JP
- **状態**: ✅ 統一済み

### Hero セクション
- **公式サイト**: 
  - 背景: アイソメトリック都市イラスト（3D風）
  - テキスト: 「人とまちと社会の未来をつなぐ 不動産コンサルティング」
  - ボタン: 「サービスを見る」「無料相談する」

- **LP**: 
  - 背景: 「海の上の都市」イラスト（16:9）
  - テキスト: 「老後・相続・不動産 窓口ひとつで解決。」
  - ボタン: 「まずは無料査定から」「ご相談はこちら」

- **差異**: ⚠️ **大きく異なる** - LPはより具体的なターゲット層（シニア）を意識した内容

### Services セクション
- **公式サイト**: 
  - 4つのサービスを番号付きで表示
  - 各サービスに「詳細を見る →」リンク

- **LP**: 
  - 3つの柱（老後資金、不動産、生前整理）
  - 各サービスに詳細な説明テキスト
  - 「詳しくはこちら」リンクなし（LP内で完結）

- **差異**: ⚠️ **意図的な差別化** - LPは独立したランディングページとして機能

### 不動産査定セクション
- **公式サイト**: なし
- **LP**: あり（重要な機能）
- **差異**: ✅ LPの独自機能

## 4. 改善が必要な箇所

### 優先度：高

1. **Hero セクションのテキスト**
   - 現在: 「HPをご覧の方へ：そのお悩み、ここで解決します」
   - 改善案: 公式サイトのトーンに合わせて、より統一感のあるメッセージに
   - 例: 「老後・相続・不動産 ワンストップで解決」

2. **Services セクションの順序**
   - 現在: 老後資金 → 不動産 → 生前整理
   - 公式サイト: 不動産購入・売却 → 老後資金・介護・相続 → 負動産処分 → 人材育成
   - 検討: LPの順序が意図的か確認

3. **About セクション**
   - 現在: なし
   - 検討: 公式サイトと同様の「HY Consultingについて」セクション追加の必要性

### 優先度：中

1. **ボタンのスタイル**
   - 公式サイト: 黒い丸型ボタン「無料相談する →」
   - LP: 青いボタン「お問い合わせ」
   - 改善案: より統一感を持たせる

2. **フッター構成**
   - 公式サイト: 企業情報、住所、リンク集
   - LP: 企業情報、住所、リンク集、ロゴ
   - 状態: ✅ 基本的に統一

3. **ニュース・コラムセクション**
   - 公式サイト: NEWS、COLUMN セクションあり
   - LP: なし
   - 検討: LPに追加する必要性

## 5. 統一すべき要素

✅ **既に統一されている**:
- ロゴデザイン
- 基本カラースキーム
- フォント
- 企業情報（住所、電話番号）

⚠️ **調整が必要**:
- Hero セクションのメッセージング
- Services セクションの構成
- ボタンのスタイル統一
- About セクションの追加検討

❌ **意図的に異なる**（LPの独自性）:
- 不動産査定フォーム
- より詳細なサービス説明
- FAQ セクション
- ターゲット層の明確化（シニア向け）

---

## 6. 推奨アクション

1. **Hero セクション**: メッセージングを「窓口ひとつで解決」に統一
2. **ボタンスタイル**: 公式サイトの黒い丸型ボタンを参考に、より統一感を持たせる
3. **About セクション**: 公式サイトと同様の企業紹介セクションを追加検討
4. **Services 説明**: より簡潔に、かつ詳細さを保つバランスを取る
5. **ナビゲーション**: 公式サイトとの整合性を確認
