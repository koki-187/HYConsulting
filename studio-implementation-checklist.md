# STUDIO実装チェックリスト - 不動産査定セクション

このチェックリストは、STUDIOで不動産査定セクションを実装する際に、各ステップが正しく完了しているかを確認するためのものです。各項目を完了したら、チェックマークを付けてください。

---

## 事前準備

- [ ] STUDIOエディタ（https://app.studio.design/projects/G3qbQAobOJ）にログイン済み
- [ ] 「LP New」ページを開いている
- [ ] 実装ガイド（`valuation-section-implementation-guide.md`）を手元に用意
- [ ] HTMLプロトタイプ（`valuation-section-prototype.html`）をブラウザで開いて参考にする

---

## ステップ1: セクションの作成

- [ ] ヒーローセクションとAboutセクションの間に新しいセクションを追加
- [ ] セクションに `id="valuation"` を設定
- [ ] セクションの背景色を `#FFFFFF`（白）に設定
- [ ] セクションのパディングを上下 `80px`、左右 `20px` に設定
- [ ] セクションの最大幅を `1200px` に設定
- [ ] セクションを中央揃えに設定

---

## ステップ2: 見出しと説明文の追加

- [ ] セクション内にボックスを追加（見出しと説明文用）
- [ ] 見出しテキスト「データベース査定（簡易）」を追加
  - [ ] フォントサイズ: `32px`
  - [ ] フォントウェイト: `700`（Bold）
  - [ ] 色: `#0D274D`
  - [ ] テキスト揃え: 中央
  - [ ] マージン下: `16px`
- [ ] 説明文「公的データベースや取引データに基づく算出を想定。AI推計ではありません。」を追加
  - [ ] フォントサイズ: `16px`
  - [ ] 色: `#666666`
  - [ ] テキスト揃え: 中央
  - [ ] マージン下: `40px`

---

## ステップ3: 2列グリッドレイアウトの作成

- [ ] セクション内に新しいボックスを追加（グリッドコンテナ）
- [ ] ボックスのディスプレイを `Grid` に設定
- [ ] グリッドテンプレートカラムを `1fr 1fr`（2列）に設定
- [ ] グリッドギャップを `40px` に設定
- [ ] モバイルブレイクポイントで以下を設定:
  - [ ] グリッドテンプレートカラム: `1fr`（1列）
  - [ ] グリッドギャップ: `20px`

---

## ステップ4: 左カラムの作成（テキストコンテンツ）

- [ ] グリッドコンテナ内にボックスを追加（左カラム）
- [ ] ボックスの背景色を `#F5F5F5` に設定
- [ ] ボックスのパディングを `30px` に設定
- [ ] ボックスのボーダーラジアスを `14px` に設定
- [ ] ボックスのボックスシャドウを `0 2px 8px rgba(0,0,0,0.1)` に設定

### アイコンの追加

- [ ] ボックス内にアイコンを追加（計算機またはチャート）
- [ ] アイコンのサイズを `48px` に設定
- [ ] アイコンの色を `#008E7D` に設定
- [ ] アイコンのマージン下を `16px` に設定

### テキストコンテンツの追加

- [ ] 見出し「入力は最小限でOK」を追加
  - [ ] フォントサイズ: `20px`
  - [ ] フォントウェイト: `700`（Bold）
  - [ ] 色: `#0D274D`
  - [ ] マージン下: `12px`
- [ ] 説明文「住所は町名レベルでも構いません。まずは概算の目安を確認できます。」を追加
  - [ ] フォントサイズ: `14px`
  - [ ] 色: `#666666`
  - [ ] 行間: `1.6`
  - [ ] マージン下: `16px`
- [ ] 注意書き「※ 実装時は、ここに「データベース査定ウィジェット（iframe）」を埋め込みます。」を追加
  - [ ] フォントサイズ: `12px`
  - [ ] 色: `#999999`
  - [ ] 背景色: `#FFF9E6`
  - [ ] パディング: `10px`
  - [ ] ボーダーラジアス: `6px`

---

## ステップ5: 右カラムの作成（デモ用フォーム）

- [ ] グリッドコンテナ内にボックスを追加（右カラム）
- [ ] ボックスの背景色を `#FFFFFF` に設定
- [ ] ボックスのパディングを `30px` に設定
- [ ] ボックスのボーダーを `1px solid #EAEAEA` に設定
- [ ] ボックスのボーダーラジアスを `14px` に設定
- [ ] ボックスのボックスシャドウを `0 2px 8px rgba(0,0,0,0.1)` に設定

### フォームフィールド1: エリア

- [ ] ラベル「エリア」を追加
  - [ ] フォントサイズ: `14px`
  - [ ] フォントウェイト: `600`（Semi-Bold）
  - [ ] 色: `#0D274D`
  - [ ] マージン下: `8px`
- [ ] セレクトボックスを追加
  - [ ] オプション: 「横浜」「湘南」「その他」
  - [ ] 高さ: `44px`
  - [ ] ボーダー: `1px solid #EAEAEA`
  - [ ] ボーダーラジアス: `8px`
  - [ ] パディング: `0 12px`
  - [ ] フォントサイズ: `14px`
  - [ ] マージン下: `20px`

### フォームフィールド2: 物件種別

- [ ] ラベル「物件種別」を追加（スタイルは上記と同じ）
- [ ] セレクトボックスを追加
  - [ ] オプション: 「戸建て」「マンション」「アパート（収益）」「土地」
  - [ ] スタイル設定は上記と同じ

### フォームフィールド3: 状況（任意）

- [ ] ラベル「状況（任意）」を追加（スタイルは上記と同じ）
- [ ] テキストエリアを追加
  - [ ] プレースホルダー: 「例：相続、共有名義、分割協議中、空き家、売却相談など」
  - [ ] 高さ: `80px`
  - [ ] ボーダー: `1px solid #EAEAEA`
  - [ ] ボーダーラジアス: `8px`
  - [ ] パディング: `12px`
  - [ ] フォントサイズ: `14px`
  - [ ] マージン下: `20px`

### CTAボタン

- [ ] ボタン「概算を確認（デモ）」を追加
  - [ ] 幅: `100%`
  - [ ] 高さ: `48px`
  - [ ] 背景色: `#008E7D`
  - [ ] テキスト色: `#FFFFFF`
  - [ ] フォントサイズ: `16px`
  - [ ] フォントウェイト: `700`（Bold）
  - [ ] ボーダーラジアス: `14px`
  - [ ] ホバー時の背景色: `#006B5E`
  - [ ] マージン下: `12px`
- [ ] 注意書き「このボタンはデモです。STUDIO実装時はウィジェットに置き換えます。」を追加
  - [ ] フォントサイズ: `12px`
  - [ ] 色: `#999999`
  - [ ] テキスト揃え: 中央

## ステップ6: レスポンシブ対応の確認

- [ ] デスクトップサイズ（1440px）で表示を確認
  - [ ] 2列グリッドが正しく表示されている
  - [ ] 左右のカラムが均等に配置されている
  - [ ] フォントサイズが適切
- [ ] タブレットサイズ（900px）で表示を確認
  - [ ] レイアウトが崩れていない
  - [ ] 必要に応じて1列に切り替わる
- [ ] モバイルサイズ（540px）で表示を確認
  - [ ] 1列グリッドに切り替わっている
  - [ ] パディングが適切（上下 `40px`、左右 `16px`）
  - [ ] フォントサイズが適切（見出し `28px`、説明文 `14px`）
  - [ ] フォームフィールドが正しく表示されている
  - [ ] ボタンが全幅で表示されている

## 追加の確認事項

- [ ] セクション内のすべての要素が既存のグローバルスタイルと統一されている
- [ ] フォントファミリーが統一されている
- [ ] カラーパレットが統一されている（メインカラー: `#0D274D`、サブカラー1: `#008E7D`）
- [ ] iOS対応: 入力フォームのフォントサイズが16px以上（自動ズーム防止）

## トラブルシューティング

### セレクトボックスが追加できない場合

- [ ] STUDIOの「フォーム」機能を使用していることを確認
- [ ] フォームコンポーネントが正しく配置されているか確認

### グリッドレイアウトが崩れる場合

- [ ] グリッドコンテナの「ディスプレイ」が `Grid` に設定されているか確認
- [ ] グリッドテンプレートカラムが正しく設定されているか確認

### モバイル表示で1列にならない場合

- [ ] レスポンシブ設定で、モバイルブレイクポイントのグリッドテンプレートカラムを `1fr` に設定
