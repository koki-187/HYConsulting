# 公式HPロゴサイズ調査結果

## 調査日時
2026-01-10

## 調査対象
公式HP: https://hyconsulting.jp/

## Headerロゴの観察結果
- スクリーンショットから確認したHeaderロゴのサイズ:
  - ロゴは左上に配置されている
  - テキスト「HY Consulting」とロゴマークが横並びで表示
  - 視覚的には小さめのサイズで控えめな印象
  - 推定サイズ: 高さ約40-50px程度

## Footerロゴの観察結果
- スクリーンショットから確認したFooterロゴのサイズ:
  - 左側に「株式会社HY」のロゴマークとテキストが表示
  - Headerロゴと同じデザイン・サイズ感
  - 視覚的には小さめのサイズで控えめな印象
  - 推定サイズ: 高さ約40-50px程度（Headerと同等）

## 現在のLPロゴサイズの確認結果

### Header.tsx
- ロゴサイズ: `h-8 lg:h-10` (32px / 40px)
- レスポンシブ設定あり

### Footer.tsx
- ロゴサイズ: `h-10` (40px)
- インラインスタイル: `width: '174px', height: '60px'`
- インラインスタイルが優先されるため、Tailwindの`h-10`は無効化されている

### Tokushoho.tsx (Header)
- ロゴサイズ: `h-8 lg:h-10` (32px / 40px)
- Header.tsxと同じ設定

### Tokushoho.tsx (Footer)
- ロゴサイズ: `h-12` (48px)
- インラインスタイル: `width: '189px', height: '148px'`
- インラインスタイルが優先されるため、Tailwindの`h-12`は無効化されている

## 問題点
1. **Footer.tsxとTokushoho.tsxのロゴサイズが不統一**
   - Footer.tsx: 174px × 60px
   - Tokushoho.tsx Footer: 189px × 148px
   - サイズが大きく異なる

2. **インラインスタイルがTailwind CSSを上書き**
   - インラインスタイルを削除し、Tailwind CSSのみで統一する必要がある

3. **公式HPのロゴサイズとの比較**
   - 公式HP: 推定40-50px程度
   - 現在のLP Header: 32px / 40px (公式HPと同等)
   - 現在のLP Footer: 60px / 148px (公式HPより大きい)

## 修正方針
1. 公式HPのロゴサイズを`h-10` (40px)に統一
2. 全てのロゴを`h-10` (40px)に設定
3. インラインスタイルを全て削除
4. レスポンシブ設定は保持（モバイルでは`h-8`、デスクトップでは`h-10`）
