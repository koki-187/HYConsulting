# 不動産査定セクション実装ガイド（STUDIO用）

## 目的
HYコンサルティングのLP Newページに、デモ用の不動産査定フォームを含む「データベース査定セクション」を追加します。このセクションは、将来的に実際の不動産査定ウィジェット（iframe）に置き換え可能な設計になっています。

## セクション概要

### デザイン
- **レイアウト**: 2列グリッド（PC）、1列（モバイル）
- **背景色**: 白（#FFFFFF）
- **余白**: 上下80px、左右20px

### コンテンツ
- **見出し**: 「データベース査定（簡易）」
- **説明文**: 「公的データベースや取引データに基づく算出を想定。AI推計ではありません。」
- **左カラム**: テキストコンテンツとアイコン
- **右カラム**: デモ用フォーム

## STUDIOでの実装手順

### ステップ1: セクションの作成

1. STUDIOエディタで「LP New」ページを開く
2. ヒーローセクションとAboutセクションの間に新しいセクションを追加
   - 左サイドバーの「追加」タブをクリック
   - 「セクション」を選択
   - セクションをキャンバスにドラッグ&ドロップ
3. セクションに `id="valuation"` を設定
   - セクションを選択
   - 右パネルの「設定」タブをクリック
   - 「ID」フィールドに `valuation` と入力

### ステップ2: セクションのスタイル設定

1. セクションを選択
2. 右パネルの「スタイル」タブをクリック
3. 以下の設定を適用:
   - **背景色**: `#FFFFFF`（白）
   - **パディング**: 上下 `80px`、左右 `20px`
   - **最大幅**: `1200px`
   - **中央揃え**: ON

### ステップ3: 見出しと説明文の追加

1. セクション内にボックスを追加
   - 左サイドバーの「追加」タブをクリック
   - 「ボックス」を選択
   - ボックスをセクション内にドラッグ&ドロップ
2. ボックス内にテキストコンポーネントを追加（見出し）
   - 左サイドバーの「追加」タブをクリック
   - 「テキスト」を選択
   - テキストをボックス内にドラッグ&ドロップ
   - テキスト内容: 「データベース査定（簡易）」
   - スタイル設定:
     - **フォントサイズ**: `32px`
     - **フォントウェイト**: `700`（Bold）
     - **色**: `#0D274D`（メインカラー）
     - **テキスト揃え**: 中央
     - **マージン**: 下 `16px`
3. ボックス内にテキストコンポーネントを追加（説明文）
   - テキスト内容: 「公的データベースや取引データに基づく算出を想定。AI推計ではありません。」
   - スタイル設定:
     - **フォントサイズ**: `16px`
     - **色**: `#666666`
     - **テキスト揃え**: 中央
     - **マージン**: 下 `40px`

### ステップ4: 2列グリッドレイアウトの作成

1. セクション内に新しいボックスを追加（グリッドコンテナ）
2. ボックスのスタイル設定:
   - **ディスプレイ**: `Grid`
   - **グリッドテンプレートカラム**: `1fr 1fr`（2列）
   - **グリッドギャップ**: `40px`
   - **レスポンシブ設定（モバイル）**:
     - **グリッドテンプレートカラム**: `1fr`（1列）
     - **グリッドギャップ**: `20px`

### ステップ5: 左カラムの作成（テキストコンテンツ）

1. グリッドコンテナ内にボックスを追加（左カラム）
2. ボックスのスタイル設定:
   - **背景色**: `#F5F5F5`（サポートカラー）
   - **パディング**: `30px`
   - **ボーダーラジアス**: `14px`
   - **ボックスシャドウ**: `0 2px 8px rgba(0,0,0,0.1)`
3. ボックス内にアイコンを追加
   - 左サイドバーの「追加」タブをクリック
   - 「アイコン」を選択
   - アイコンをボックス内にドラッグ&ドロップ
   - アイコン: 「計算機」または「チャート」
   - スタイル設定:
     - **サイズ**: `48px`
     - **色**: `#008E7D`（サブカラー1）
     - **マージン**: 下 `16px`
4. ボックス内にテキストコンポーネントを追加（見出し）
   - テキスト内容: 「入力は最小限でOK」
   - スタイル設定:
     - **フォントサイズ**: `20px`
     - **フォントウェイト**: `700`（Bold）
     - **色**: `#0D274D`
     - **マージン**: 下 `12px`
5. ボックス内にテキストコンポーネントを追加（説明文）
   - テキスト内容: 「住所は町名レベルでも構いません。まずは概算の目安を確認できます。」
   - スタイル設定:
     - **フォントサイズ**: `14px`
     - **色**: `#666666`
     - **行間**: `1.6`
     - **マージン**: 下 `16px`
6. ボックス内にテキストコンポーネントを追加（注意書き）
   - テキスト内容: 「※ 実装時は、ここに「データベース査定ウィジェット（iframe）」を埋め込みます。」
   - スタイル設定:
     - **フォントサイズ**: `12px`
     - **色**: `#999999`
     - **背景色**: `#FFF9E6`（薄い黄色）
     - **パディング**: `10px`
     - **ボーダーラジアス**: `6px`

### ステップ6: 右カラムの作成（デモ用フォーム）

1. グリッドコンテナ内にボックスを追加（右カラム）
2. ボックスのスタイル設定:
   - **背景色**: `#FFFFFF`（白）
   - **パディング**: `30px`
   - **ボーダー**: `1px solid #EAEAEA`
   - **ボーダーラジアス**: `14px`
   - **ボックスシャドウ**: `0 2px 8px rgba(0,0,0,0.1)`

#### フォームフィールド1: エリア

1. ボックス内にテキストコンポーネントを追加（ラベル）
   - テキスト内容: 「エリア」
   - スタイル設定:
     - **フォントサイズ**: `14px`
     - **フォントウェイト**: `600`（Semi-Bold）
     - **色**: `#0D274D`
     - **マージン**: 下 `8px`
2. ボックス内にセレクトボックスを追加
   - 左サイドバーの「追加」タブをクリック
   - 「フォーム」→「セレクト」を選択
   - セレクトボックスをボックス内にドラッグ&ドロップ
   - オプション:
     - 「横浜」
     - 「湘南」
     - 「その他」
   - スタイル設定:
     - **高さ**: `44px`
     - **ボーダー**: `1px solid #EAEAEA`
     - **ボーダーラジアス**: `8px`
     - **パディング**: `0 12px`
     - **フォントサイズ**: `14px`
     - **マージン**: 下 `20px`

#### フォームフィールド2: 物件種別

1. ボックス内にテキストコンポーネントを追加（ラベル）
   - テキスト内容: 「物件種別」
   - スタイル設定: 上記と同じ
2. ボックス内にセレクトボックスを追加
   - オプション:
     - 「戸建て」
     - 「マンション」
     - 「アパート（収益）」
     - 「土地」
   - スタイル設定: 上記と同じ

#### フォームフィールド3: 状況（任意）

1. ボックス内にテキストコンポーネントを追加（ラベル）
   - テキスト内容: 「状況（任意）」
   - スタイル設定: 上記と同じ
2. ボックス内にテキストエリアを追加
   - 左サイドバーの「追加」タブをクリック
   - 「フォーム」→「テキストエリア」を選択
   - テキストエリアをボックス内にドラッグ&ドロップ
   - プレースホルダー: 「例：相続、共有名義、分割協議中、空き家、売却相談など」
   - スタイル設定:
     - **高さ**: `80px`
     - **ボーダー**: `1px solid #EAEAEA`
     - **ボーダーラジアス**: `8px`
     - **パディング**: `12px`
     - **フォントサイズ**: `14px`
     - **マージン**: 下 `20px`

#### CTAボタン

1. ボックス内にボタンコンポーネントを追加
   - 左サイドバーの「追加」タブをクリック
   - 「ボタン」を選択
   - ボタンをボックス内にドラッグ&ドロップ
   - テキスト内容: 「概算を確認（デモ）」
   - スタイル設定:
     - **幅**: `100%`
     - **高さ**: `48px`
     - **背景色**: `#008E7D`（サブカラー1）
     - **テキスト色**: `#FFFFFF`（白）
     - **フォントサイズ**: `16px`
     - **フォントウェイト**: `700`（Bold）
     - **ボーダーラジアス**: `14px`
     - **ホバー時の背景色**: `#006B5E`（暗いサブカラー1）
     - **マージン**: 下 `12px`
2. ボックス内にテキストコンポーネントを追加（注意書き）
   - テキスト内容: 「このボタンはデモです。STUDIO実装時はウィジェットに置き換えます。」
   - スタイル設定:
     - **フォントサイズ**: `12px`
     - **色**: `#999999`
     - **テキスト揃え**: 中央

### ステップ7: レスポンシブ対応の確認

1. STUDIOエディタの上部にあるデバイスサイズ切り替えボタンをクリック
2. 以下のサイズで表示を確認:
   - **デスクトップ**: 1440px（2列グリッド）
   - **タブレット**: 900px（2列グリッドまたは1列）
   - **モバイル**: 540px（1列グリッド）
3. モバイルサイズで以下の調整を行う:
   - グリッドテンプレートカラム: `1fr`（1列）
   - パディング: 上下 `40px`、左右 `16px`
   - フォントサイズ: 見出し `28px`、説明文 `14px`

### ステップ8: セクションの位置調整

1. レイヤーパネルで「valuation」セクションを選択
2. ヒーローセクションとAboutセクションの間に配置されていることを確認
3. 必要に応じて、ドラッグ&ドロップで位置を調整

## 将来的なウィジェット置き換え

不動産査定ウィジェットのURLが提供されたら、以下の手順で置き換えます。

1. 右カラム（デモ用フォーム）を削除
2. 右カラムに「Embed Box」を追加
   - 左サイドバーの「追加」タブをクリック
   - 「Embed」を選択
   - Embed Boxを右カラムにドラッグ&ドロップ
3. Embed Boxに以下のiframeコードを貼り付け:
   ```html
   <iframe 
     src="ウィジェットURL" 
     width="100%" 
     height="500px" 
     frameborder="0" 
     style="border-radius: 14px; box-shadow: 0 2px 8px rgba(0,0,0,0.1);"
   ></iframe>
   ```
4. ウィジェットURLを実際のURLに置き換え
5. 高さ（`height`）を適切な値に調整

## トラブルシューティング

### セレクトボックスが追加できない場合
- STUDIOの「フォーム」機能を使用していることを確認
- フォームコンポーネントが正しく配置されているか確認

### グリッドレイアウトが崩れる場合
- グリッドコンテナの「ディスプレイ」が `Grid` に設定されているか確認
- グリッドテンプレートカラムが正しく設定されているか確認

### モバイル表示で1列にならない場合
- レスポンシブ設定で、モバイルブレイクポイントのグリッドテンプレートカラムを `1fr` に設定

## まとめ

この実装ガイドに従うことで、HYコンサルティングのLP Newページに、デモ用の不動産査定フォームを含む「データベース査定セクション」を追加できます。将来的に実際の不動産査定ウィジェットのURLが提供されたら、デモ用フォームを簡単にiframeに置き換えることができます。

実装中に問題が発生した場合は、トラブルシューティングセクションを参照してください。
