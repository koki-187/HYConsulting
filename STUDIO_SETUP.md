# HYConsulting LP - STUDIO セットアップガイド

## プロジェクト概要
HYConsultingのランディングページをSTUDIOで構築するプロジェクトです。

## STUDIOプロジェクト情報
- **プロジェクトURL**: https://app.studio.design/projects/G3qbQAobOJ/dashboard/home/general
- **ヘルプドキュメント**: https://help.studio.design/ja/

## セットアップ手順

### 1. STUDIOアカウントへのアクセス
1. 上記のプロジェクトURLにアクセス
2. STUDIOアカウントでログイン
3. プロジェクトダッシュボードを確認

### 2. プロジェクト構成
このリポジトリでは以下を管理します：
- LPのコンテンツ仕様
- デザインガイドライン
- アセット（画像、テキストなど）
- 公開手順のドキュメント

### 3. STUDIO の主な機能
- ノーコードでWebサイト構築
- レスポンシブデザイン対応
- CMS機能
- フォーム機能
- アニメーション機能
- SEO設定

## 開発フロー

### STUDIOでの作業フロー
1. デザイン仕様の確認（`docs/design/`）
2. STUDIOエディタでページを構築
3. プレビューで確認
4. 公開設定の確認
5. 本番環境へ公開

### このリポジトリの役割
- コンテンツの管理
- 仕様書の管理
- アセットの管理
- バージョン管理
- ドキュメント管理
