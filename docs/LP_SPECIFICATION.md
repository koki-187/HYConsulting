# HYConsulting ランディングページ 仕様書

## ページ構成

### 1. ファーストビュー（FV）
- **キャッチコピー**: メインメッセージ
- **サブコピー**: 補足説明
- **CTA（Call To Action）**: お問い合わせボタン
- **ヒーローイメージ**: メインビジュアル

### 2. サービス紹介セクション
- **見出し**: 提供サービスの概要
- **サービス一覧**: 
  - サービス1（アイコン + 説明）
  - サービス2（アイコン + 説明）
  - サービス3（アイコン + 説明）

### 3. 強み・特徴セクション
- **見出し**: HYConsultingの強み
- **特徴リスト**:
  - 特徴1（イメージ + タイトル + 説明）
  - 特徴2（イメージ + タイトル + 説明）
  - 特徴3（イメージ + タイトル + 説明）

### 4. 実績・事例セクション
- **見出し**: 導入実績
- **実績カード**: クライアント事例の紹介
- **数値実績**: 実績を数字で表示

### 5. お客様の声セクション
- **見出し**: お客様の声
- **レビューカード**: 顧客レビュー（3-5件）
  - 顧客名
  - 所属
  - コメント
  - 評価

### 6. 料金プランセクション（オプション）
- **見出し**: 料金プラン
- **プランカード**: 各プランの詳細
  - プラン名
  - 価格
  - 含まれる内容
  - CTAボタン

### 7. よくある質問（FAQ）
- **見出し**: よくある質問
- **Q&Aリスト**: アコーディオン形式
  - 質問1と回答
  - 質問2と回答
  - 質問3と回答

### 8. お問い合わせセクション
- **見出し**: お問い合わせ
- **フォーム**: 
  - 氏名
  - メールアドレス
  - 会社名
  - お問い合わせ内容
  - 送信ボタン
- または外部フォームへのリンク

### 9. フッター
- **企業情報**: 
  - 会社名
  - 住所
  - 連絡先
- **リンク**: 
  - プライバシーポリシー
  - 利用規約
- **SNSリンク**: ソーシャルメディアアイコン
- **コピーライト**: © 2024 HYConsulting

## デザイン要件

### カラースキーム
- **プライマリカラー**: #0066CC（青系）
- **セカンダリカラー**: #00CC66（緑系）
- **アクセントカラー**: #FF6600（オレンジ系）
- **ベースカラー**: #FFFFFF（白）
- **テキストカラー**: #333333（濃いグレー）

### タイポグラフィ
- **見出しフォント**: ゴシック体（太字）
- **本文フォント**: ゴシック体（レギュラー）
- **フォントサイズ**:
  - H1: 48px
  - H2: 36px
  - H3: 24px
  - 本文: 16px

### レスポンシブ対応
- **デスクトップ**: 1200px以上
- **タブレット**: 768px - 1199px
- **モバイル**: 767px以下

## 機能要件

### アニメーション
- スクロールアニメーション（フェードイン）
- ホバーエフェクト（ボタン、カード）
- スムーススクロール

### SEO設定
- **タイトル**: HYConsulting | [サービス説明]
- **メタディスクリプション**: 150-160文字
- **OGPタグ**: SNSシェア用画像とテキスト
- **構造化データ**: Organization, WebSite

### パフォーマンス
- **画像最適化**: WebP形式推奨
- **ローディング**: 3秒以内
- **Lighthouse スコア**: 90点以上目標

## 公開設定
- **独自ドメイン**: 設定予定のドメイン
- **SSL証明書**: 有効化
- **アナリティクス**: Google Analytics連携
- **サイトマップ**: 自動生成

## 更新フロー
1. STUDIOエディタで編集
2. プレビューで確認
3. 公開ボタンで本番反映
4. 変更内容をこのリポジトリに記録
