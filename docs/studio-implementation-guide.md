## 実装の準備

### STEP 1: グローバルスタイルの設定

STUDIOでは、サイト全体で使用するカラー、フォント、スタイルをグローバルに設定できます。まず最初に、デザインシステムに基づいてグローバルスタイルを設定します。

#### カラーパレットの登録

1. エディタ左サイドバーの「デザイン」タブをクリックします。
2. 「カラーパレット」セクションを開きます。
3. 以下のカラーを登録します。

| カラー名 | HEX | 用途 |
|:---|:---|:---|
| メインカラー | `#0D274D` | ヘッダー、フッター、見出し |
| サブカラー1 | `#008E7D` | CTAボタン、リンク |
| サブカラー2 | `#4EBFA7` | 背景、アクセント |
| サポートカラー | `#F5F5F5` | 背景セクション |
| ワームグレー | `#EAEAEA` | テキスト補足、ボーダー |
| コーラルアクセント | `#F7B042` | 強調要素 |

#### タイポグラフィの設定

1. 「デザイン」タブで「タイポグラフィ」セクションを開きます。
2. 以下のテキストスタイルを登録します。

| スタイル名 | フォント | サイズ | ウェイト | 行間 |
|:---|:---|:---|:---|:---|
| H1 | Noto Sans JP | 36px | SemiBold | 1.3 |
| H2 | Noto Sans JP | 28px | Bold | 1.4 |
| H3 | Noto Sans JP | 22px | Medium | 1.4 |
| 本文（大） | Noto Sans JP | 16px | Regular | 1.5 |
| 本文（小） | Noto Sans JP | 14px | Regular | 1.5 |

### STEP 2: アセットのアップロード

実装に必要な画像やアイコンをSTUDIOにアップロードします。

1. エディタ左サイドバーの「アセット」タブをクリックします。
2. 「アップロード」ボタンをクリックして、以下の画像をアップロードします。
   - ロゴ画像
   - ヒーローセクション背景画像（3Dイラスト）
   - サービスアイコン（4種類）
   - 実績・事例の画像
   - スタッフ写真

3. アップロード後、各画像に適切な名前を付けて管理しやすくします。

## ページ構築

### STEP 3: ヘッダーの作成

ヘッダーは全ページで共通して表示される重要な要素です。固定ヘッダーとして設定し、スクロールしても常に表示されるようにします。

#### 構成要素

- **ロゴ（左）**: HYコンサルティングのロゴを配置します。
- **ナビゲーション（中央）**: 「サービス」「事例」「FAQ」「お問い合わせ」のリンクを配置します。
- **CTAボタン（右）**: 「無料会員登録」「無料相談」のボタンを配置します。

#### 実装手順

1. 新規セクションを追加し、「固定ヘッダー」オプションを有効にします。
2. セクションの高さを80pxに設定します。
3. 背景色を白（`#FFFFFF`）または半透明に設定します。
4. 12カラムグリッドを使用してレイアウトを構成します。
   - 左3カラム: ロゴ
   - 中央6カラム: ナビゲーションメニュー
   - 右3カラム: CTAボタン
5. z-indexを1000に設定して、他の要素より前面に表示されるようにします。

#### モバイル対応

モバイル表示では、ハンバーガーメニューを使用してナビゲーションを折りたたみます。

1. デバイス切り替えで「スマートフォン」を選択します。
2. ナビゲーションメニューを非表示にし、ハンバーガーアイコンを配置します。
3. ハンバーガーアイコンをクリックすると、メニューがスライドインで表示されるように設定します。

### STEP 4: ヒーローセクションの作成

ヒーローセクションは、LPの第一印象を決定づける最も重要なセクションです。ユーザーの注意を引き、行動を促すデザインにします。

#### 構成要素

- **H1見出し**: 「人とまちと社会の未来をつなぐ不動産コンサルティング」
- **サブコピー**: 「理想の住まいづくりや資産形成など「未来のビジョン」から、空き家・相続といった「いまの課題」まで、不動産を活用したソリューションを提案し実行支援していきます」
- **CTAボタン**: 「サービスを見る」「無料相談する」の2つのボタン
- **背景**: 3Dイラスト（都市建築イメージ）

#### 実装手順

1. 新規セクションを追加します。
2. セクションの高さを100vh（画面いっぱい）に設定します。
3. 背景画像として3Dイラストをアップロードし、適用します。
4. 背景画像の上に半透明のオーバーレイを追加して、テキストの可読性を向上させます。
5. H1見出し、サブコピー、CTAボタンを配置します。
6. テキストは左寄せまたは中央揃えにし、適切な余白を確保します。
7. CTAボタンには、サブカラー1（`#008E7D`）を適用し、ホバーエフェクトを設定します。

#### アニメーション

ヒーローセクションの要素にフェードインアニメーションを追加して、視覚的なインパクトを高めます。

1. H1見出しを選択し、「アニメーション」タブを開きます。
2. 「ページ読み込み時」を選択し、「フェードイン」を適用します。
3. 遅延時間を0.2秒に設定します。
4. サブコピーとCTAボタンにも同様にアニメーションを追加し、遅延時間を0.4秒、0.6秒と段階的に設定します。

### STEP 5: Aboutセクションの作成

Aboutセクションでは、HYコンサルティングの理念とサービス概要を紹介します。

#### 構成要素

- **H2見出し**: 「HY Consultingについて」
- **本文**: 会社の理念とサービス概要
- **ビジュアル**: ビジネスマンが握手するイラスト
- **CTAボタン**: 「会社情報を見る」

#### 実装手順

1. 新規セクションを追加します。
2. 2カラムレイアウトを作成します。
   - 左カラム: テキスト（H2見出し、本文、CTAボタン）
   - 右カラム: イラスト画像
3. 背景色をサポートカラー（`#F5F5F5`）に設定します。
4. 適切な余白（上下80px、左右24px）を確保します。

### STEP 6: Servicesセクションの作成

Servicesセクションでは、HYコンサルティングが提供する4つの主要サービスを紹介します。

#### 構成要素

- **H2見出し**: 「個人から法人まで幅広い課題に対応する HY Consultingのサービス」
- **サービスカード**: 4つのサービスをカード形式で配置

#### 実装手順

1. 新規セクションを追加します。
2. H2見出しを配置します。
3. 2x2のグリッドレイアウトを作成し、各セルにサービスカードを配置します。
4. 各カードには以下の要素を含めます。
   - 番号（01, 02, 03, 04）
   - サービスタイトル
   - サービス説明
   - アイコンまたはイラスト
5. カードにホバーエフェクトを追加します（影の強調、わずかな拡大など）。

#### モバイル対応

モバイル表示では、カードを1カラムで縦に並べます。

### STEP 7: Columnセクションの作成

Columnセクションでは、お役立ち情報の記事を紹介します。

#### 構成要素

- **H2見出し**: 「お役立ち情報」
- **記事カード**: 複数の記事をカード形式で配置
- **CTAボタン**: 「一覧を見る」

#### 実装手順

1. 新規セクションを追加します。
2. H2見出しを配置します。
3. 3カラムレイアウトを作成し、各カラムに記事カードを配置します。
4. 各カードには以下の要素を含めます。
   - サムネイル画像
   - カテゴリラベル
   - 記事タイトル
   - 公開日
5. セクションの下部に「一覧を見る」ボタンを配置します。

### STEP 8: フッターの作成

フッターには、会社情報、リンク、SNSアイコン、コピーライトを配置します。

#### 構成要素

- **会社情報**: 会社名、住所、連絡先
- **リンク**: プライバシーポリシー、利用規約
- **SNSアイコン**: ソーシャルメディアへのリンク
- **コピーライト**: © 2025 HY Consulting. All Rights Reserved.

#### 実装手順

1. 新規セクションを追加します。
2. 背景色をメインカラー（`#0D274D`）に設定します。
3. テキスト色を白（`#FFFFFF`）に設定します。
4. 3カラムレイアウトを作成し、各カラムに情報を配置します。
5. 最下部にコピーライトを中央揃えで配置します。

## レスポンシブ対応

STUDIOでは、デバイスごとに異なるレイアウトを設定できます。以下のブレイクポイントで最適化を行います。

| デバイス | 画面幅 | 調整内容 |
|:---|:---|:---|
| **デスクトップ** | 1200px以上 | 12カラムグリッド、複数カラムレイアウト |
| **タブレット** | 768px～1199px | 8カラムグリッド、一部カラム数を減らす |
| **スマートフォン** | 767px以下 | 4カラムグリッド、1カラムレイアウト |

### レスポンシブ調整のポイント

1. **フォントサイズ**: モバイルでは見出しと本文のサイズを若干小さくします。
2. **余白**: モバイルでは余白を狭めて、コンテンツ領域を広く確保します。
3. **画像**: モバイルでは画像を縦に並べ、横幅いっぱいに表示します。
4. **ボタン**: モバイルでは横幅いっぱいのボタンにして、タップしやすくします。

### 構造化データの追加

FAQPageスキーマとLocalBusinessスキーマをJSON-LD形式で追加します。

1. 「設定」→「カスタムコード」を開きます。
2. 「ヘッド内」に以下のJSON-LDコードを追加します。

```json
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "HY Consulting",
  "image": "https://hyconsulting.jp/images/logo.png",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "横浜市",
    "addressRegion": "神奈川県",
    "addressCountry": "JP"
  },
  "telephone": "+81-XX-XXXX-XXXX",
  "url": "https://hyconsulting.jp"
}
</script>
```

## プレビューと公開

### プレビュー

1. エディタ右上の「プレビュー」ボタンをクリックします。
2. 各デバイスサイズで表示を確認します。
3. リンク、ボタン、フォームの動作を確認します。

### 公開

1. 「公開」ボタンをクリックします。
2. 公開設定を確認します。
   - ドメイン設定
   - SSL設定
3. 「公開する」をクリックします。

### 独自ドメイン設定

1. 「設定」→「ドメイン」を開きます。
2. 「カスタムドメインを追加」をクリックします。
3. ドメインを入力します（例: hyconsulting.jp）。
4. DNS設定を行います（CNAMEレコードを追加）。
5. SSL証明書の発行を待ちます（通常24時間以内）。

## トラブルシューティング

### よくある問題と解決方法

| 問題 | 解決方法 |
|:---|:---|
| **レイアウトが崩れる** | ボックスのサイズ設定を確認し、レスポンシブ設定を見直します。z-indexの競合も確認します。 |
| **アニメーションが動作しない** | アニメーション設定を再確認し、スクロールトリガーの位置を調整します。ブラウザのキャッシュをクリアします。 |
| **フォームが送信できない** | 送信先メールアドレスを確認し、必須項目の設定を確認します。reCAPTCHA設定も確認します。 |
| **画像が表示されない** | 画像のファイル形式とサイズを確認します。WebP形式を推奨します。 |
