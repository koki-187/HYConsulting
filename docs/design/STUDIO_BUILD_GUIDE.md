# STUDIOでのLP構築ガイド

## はじめに
このガイドでは、STUDIOを使用してHYConsultingのランディングページを構築する手順を説明します。

## STUDIOプロジェクトへのアクセス

### ログイン方法
1. https://app.studio.design/projects/G3qbQAobOJ/dashboard/home/general にアクセス
2. STUDIOアカウントでログイン
3. HYConsultingプロジェクトのダッシュボードが表示されます

## STUDIO基本操作

### エディタの起動
1. ダッシュボードで「エディタを開く」をクリック
2. デザインキャンバスが表示されます

### ページの作成
1. 左サイドバーの「ページ」タブをクリック
2. 「+新規ページ」をクリック
3. ページ名を入力（例: index, about, contactなど）
4. ページタイプを選択（通常ページ/テンプレートページ）

### セクションの追加
1. 左サイドバーの「追加」タブをクリック
2. 「セクション」を選択
3. セクションをキャンバスにドラッグ&ドロップ
4. セクション内にボックスを配置してレイアウトを構成

### コンポーネントの配置
STUDIOで使用できる主なコンポーネント：
- **テキスト**: 見出し、段落、リストなど
- **画像**: 静止画、背景画像
- **ボタン**: CTAボタン、リンクボタン
- **フォーム**: お問い合わせフォーム
- **ボックス**: レイアウト用のコンテナ
- **動画**: 埋め込み動画
- **アイコン**: 装飾用アイコン

## LP構築ステップ

### STEP 1: ヘッダーの作成
1. 新規セクションを追加（固定ヘッダー設定）
2. ロゴを配置（画像コンポーネント）
3. ナビゲーションメニューを追加
4. CTAボタンを配置

```
推奨設定:
- セクション高さ: 80px
- 背景: 白または半透明
- 固定ヘッダー: ON
- z-index: 1000
```

### STEP 2: ファーストビューの作成
1. 新規セクションを追加
2. 背景画像またはグラデーションを設定
3. キャッチコピー（H1）を配置
4. サブコピー（段落）を配置
5. CTAボタンを配置

```
推奨設定:
- セクション高さ: 100vh（画面いっぱい）
- テキスト配置: 中央または左寄せ
- アニメーション: フェードイン
```

### STEP 3: サービス紹介セクション
1. 3カラムレイアウトを作成
2. 各カラムにアイコン、タイトル、説明を配置
3. ホバーエフェクトを設定

```
レイアウト:
[アイコン] [アイコン] [アイコン]
[タイトル] [タイトル] [タイトル]
[説明文] [説明文] [説明文]
```

### STEP 4: 強み・特徴セクション
1. 2カラムレイアウト（画像 + テキスト）を作成
2. 交互配置でリズムを作る
3. スクロールアニメーションを追加

```
パターン1: [画像][テキスト]
パターン2: [テキスト][画像]
パターン3: [画像][テキスト]
```

### STEP 5: 実績セクション
1. 数値を強調する大きなフォントを使用
2. カウントアップアニメーションを検討
3. 4カラムレイアウトで実績を並べる

### STEP 6: お客様の声セクション
1. カードコンポーネントを使用
2. 3カラムレイアウト
3. 引用符デザインを追加
4. 顧客名・所属を明記

### STEP 7: FAQセクション
1. アコーディオンコンポーネントを使用
2. Q&A形式で配置
3. 開閉アニメーションを設定

### STEP 8: お問い合わせフォーム
1. フォームコンポーネントを追加
2. 入力項目を設定
   - 氏名（必須）
   - メールアドレス（必須）
   - 会社名
   - お問い合わせ内容（必須）
3. 送信先メールアドレスを設定
4. サンクスページまたはメッセージを設定

### STEP 9: フッターの作成
1. 3-4カラムレイアウト
2. 会社情報、リンク、SNSアイコンを配置
3. コピーライトを追加

## デザインの統一

### カラー設定
1. 「デザイン」タブで「カラーパレット」を開く
2. ブランドカラーを登録
   - プライマリ: #0066CC
   - セカンダリ: #00CC66
   - アクセント: #FF6600
3. コンポーネントにカラーを適用

### フォント設定
1. 「デザイン」タブで「タイポグラフィ」を開く
2. 見出しと本文のフォントを設定
3. フォントサイズのスケールを定義

### レスポンシブ対応
1. エディタ上部のデバイス切り替えボタンをクリック
2. 各デバイスサイズ（PC/タブレット/スマホ）でレイアウトを調整
3. 非表示設定を活用してモバイルでは簡略化

## アニメーション設定

### スクロールアニメーション
1. 要素を選択
2. 右サイドバーで「アニメーション」タブを開く
3. 「スクロール時」を選択
4. アニメーションタイプを選択（フェードイン、スライドインなど）
5. タイミングを調整

### ホバーエフェクト
1. ボタンやカードを選択
2. 「ホバー」状態を選択
3. 色、サイズ、影などを変更

## SEO設定

### ページ設定
1. ページリストで対象ページをクリック
2. 「ページ設定」を開く
3. 以下を設定：
   - ページタイトル（60文字以内）
   - メタディスクリプション（150-160文字）
   - OGP画像（1200x630px推奨）
   - URL slug

### サイト全体の設定
1. 「設定」→「サイト設定」を開く
2. サイトタイトル、説明、ファビコンを設定
3. Google Analyticsの連携

## プレビューと公開

### プレビュー
1. エディタ右上の「プレビュー」ボタンをクリック
2. 各デバイスサイズで表示確認
3. リンクや動作を確認

### 公開
1. 「公開」ボタンをクリック
2. 公開設定を確認
   - ドメイン設定
   - SSL設定
3. 「公開する」をクリック

### 独自ドメイン設定
1. 「設定」→「ドメイン」を開く
2. 「カスタムドメインを追加」をクリック
3. ドメインを入力
4. DNS設定を行う（CNAMEレコード）
5. SSL証明書の発行を待つ

## トラブルシューティング

### よくある問題と解決方法

#### レイアウトが崩れる
- ボックスのサイズ設定を確認
- レスポンシブ設定を見直す
- z-indexの競合を確認

#### アニメーションが動作しない
- アニメーション設定を再確認
- スクロールトリガーの位置を調整
- ブラウザのキャッシュをクリア

#### フォームが送信できない
- 送信先メールアドレスを確認
- 必須項目の設定を確認
- reCAPTCHA設定を確認
