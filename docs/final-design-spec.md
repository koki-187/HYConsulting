# HYコンサルティング ランディングページ デザイン仕様書

## 1. プロジェクト概要

本ドキュメントは、HYコンサルティングの新規ランディングページ（LP）をSTUDIOで構築するための最終的なデザイン仕様とコンテンツ構成を定義するものです。提供されたすべての参照資料、スクリーンショット、およびGoogleドライブ内のドキュメントを統合し、実装の基準となる仕様を定めます。

### 1.1. プロジェクトの目的

横浜・湘南エリアを対象に、相続不動産、空き家、その他の複雑な不動産問題に関する相談窓口として機能するLPを構築します。主な目的は、HYコンサルティングへの問い合わせ件数および会員登録数を増加させ、コンバージョン率を最大化することです。

### 1.2. 主要KPI

| 指標 | 説明 |
| :--- | :--- |
| **コンバージョン数** | LP経由での問い合わせ件数および会員登録件数。 |
| **コンバージョン率** | 既存サイトや広告LPと比較した転換率の向上。 |
| **エンゲージメント** | 直帰率の低下、平均滞在時間の増加、スクロール率の向上。 |
| **CTA到達率** | ファーストビューから主要なCTA（Call To Action）への到達ユーザー割合。 |

### 1.3. ターゲットユーザー

本LPは、以下のペルソナを主要なターゲットとして想定しています。

| ペルソナ | 詳細 |
| :--- | :--- |
| **相続オーナー** | 50～60代。親から不動産を相続したが、税金、管理、共有名義の分割に悩んでいる。特に横浜・湘南エリアに遠方の実家を持つ層。 |
| **投資家・賃貸オーナー** | 30～50代。横浜や湘南に所有するアパートやマンションの老朽化や空室率に悩み、売却や活用を検討している。 |
| **特殊事情の物件所有者** | 他の不動産会社に断られた区分所有マンションや元商店など、専門的なサポートを必要としている。 |

## 4. 機能要件

### 4.1. データベース査定機能
- **方式**: 国土交通省の実取引データ約10万件を格納したデータベースを使用。
- **実装**: 外部サーバーでホストされたReact製ウィジェットをiframeで埋め込み。
- **入力項目**: 物件種別、都道府県、市区町村、面積、築年数。
- **出力内容**: 市場分析、価値評価、将来予測の3部構成レポート。

### 4.2. フォーム機能
- **実装**: STUDIO標準フォームを使用し、Googleスプレッドシートと連携。
- **自動化**: Zapier経由で自動返信メールを送信。
- **セキュリティ**: reCAPTCHA v3を実装し、スパムを防止。

### 4.3. SEOおよびトラッキング
- **キーワード**: 「横浜」「湘南」「相続不動産」「空き家」などをタイトル、見出し、本文に自然に配置。
- **構造化データ**: FAQPageスキーマ、LocalBusinessスキーマをJSON-LD形式で実装。
- **トラッキング**: Google Analytics, Google Tag Manager, Facebook Pixelを導入。
